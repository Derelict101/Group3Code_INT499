# EZTechMovie React Cart System

This React application includes the Week 2 cart functionality for the StreamList and EZTechMovie project.

## Features

1. Navigation bar with Subscriptions and Cart links.
2. Cart item counter and running total in the navigation bar.
3. Product cards loaded from `src/data.js`.
4. One active subscription limit.
5. Accessories can be added multiple times.
6. Cart summary with item totals and price totals.
7. Quantity adjustment for accessories.
8. Remove item and clear cart options.
9. Cart persistence through local storage.
10. Responsive CSS styling.

## Run the project

```bash
npm install
npm run dev
```

Open the local Vite link shown in the terminal.
